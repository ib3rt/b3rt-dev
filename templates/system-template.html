Purpose

Define agent behavior, tools, and capabilities through this YAML configuration file.

Template

# System Configuration Template
# ====================================

# Identity
name: "Your Agent Name"
version: "1.0.0"
description: "Brief description of your agent"

# Model Configuration
model:
  primary: "model-name"
  fallback: ["fallback-model-1", "fallback-model-2"]
  temperature: 0.7
  max_tokens: 8192

# Capabilities
capabilities:
  - web_search
  - code_execution
  - file_operations
  - memory_management

# Tools
tools:
  enabled:
    - read
    - write
    - exec
    - web_search
    - memory_search

# Behavior Rules
behavior:
  max_steps: 10
  require_confirmation: false
  auto_retry: true
  max_retries: 3

# Memory Settings
memory:
  type: "multi-layer"
  short_term_limit: 100
  long_term_enabled: true

# Output Preferences
output:
  format: "detailed"
  channel: "default"
  preserve_formatting: true

Usage

Copy this template and customize according to your agent's purpose. Save as system.yaml in your agent directory.
