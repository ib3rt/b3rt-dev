Purpose

Define automated processes with triggers, actions, and conditional logic for your Life OS.

Template

# Workflow Automation Template
# ====================================

# Workflow Metadata
name: "workflow-name"
version: "1.0.0"
description: "What this workflow accomplishes"

# Trigger - When does this workflow start?
trigger:
  type: "schedule"  # Options: schedule, event, webhook, manual
  schedule: "0 9 * * 1-5"  # Cron expression (9 AM weekdays)
  # Or for events:
  event: "new_email_received"
  conditions:
    - field: "sender"
      contains: "important"

# Actions - What happens when triggered?
actions:
  - name: "action-1"
    type: "notification"
    channel: "telegram"
    message: "Morning brief ready!"

  - name: "action-2"
    type: "agent_run"
    agent: "research-agent"
    input: "Latest AI news"

  - name: "action-3"
    type: "webhook"
    url: "https://api.example.com/notify"
    method: "POST"
    body: '{"status": "complete"}'

# Conditional Logic
conditions:
  - if: "action-2.success == true"
    then:
      - name: "success-path"
        actions: ["action-3"]
    else:
      - name: "failure-path"
        actions: ["action-4"]

# Error Handling
error_handling:
  on_failure: "notify_admin"
  max_retries: 3
  retry_delay: 60  # seconds

Save as workflow.yaml and deploy to your automation system.
